This project is made using Gdevelop!
To clone this project simply:

1: Install gdevelop if not already (gdevelop.io)

2: Create a new folder in Documents/Gdevelop projects/

3: Place all files into the new folder (NOT the files from a release)

4: Open Gdevelop and Open a Project. then choose from computer 

5: Then select the .json file inside the previous new folder

6: Thats it!
